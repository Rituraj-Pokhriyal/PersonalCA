# 💼 Personal CA App — Rituraj

A comprehensive personal finance app powered by CA Sharma (50 years experience).

## Features
- 🏠 **Dashboard** — Net worth, portfolio snapshot, CA tips
- 💰 **Budget Tracker** — Monthly expenses, 50-30-20 analysis
- 💳 **Debt Planner** — Avalanche/Snowball strategies, 12-month projection
- 📈 **Investment Portfolio** — Groww stocks, mutual funds, US stocks
- 🧾 **Tax Planner** — Old vs New regime, 80C suggestions, ITR tips

## Setup

```bash
npm install
npx expo start
```

Scan QR code with Expo Go app (iOS/Android).

## Tech Stack
- React Native + Expo
- TypeScript
- React Navigation (Bottom Tabs)
- @expo/vector-icons (Ionicons)
